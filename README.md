## Lab 6 - Content Style

## Description
A CSS styling tutorial demonstrating how to style website content 
using CSS properties and design principles.

## Live Website
[Click here to view the website](https://jalilgreen11-png.github.io/lab6-content-style/)

## Topics Covered
- Typography (font-family, font-size, font-weight)
- Text alignment and centering
- The CSS Box Model (padding, margin, border)
- Flexbox layout
- Navigation bar with pseudo-classes
- Color with hexadecimal values
